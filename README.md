## 🧑‍💻 For Developers

If you are a developer joining the project, please reach the **[Developer Setup & Onboarding Guide](./DEVELOPER_GUIDE.md)** for:

- Understanding the `main` vs `development` branching strategy.
- Setting up your local XAMPP environment.
- Connecting to the live Aiven Cloud Database via HeidiSQL.
- Best practices for testing and deployment.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS, JavaScript.
- **Backend**: PHP 8.x.
- **Database**: MySQL (Hosted on Aiven).
- **Deployment**: Vercel (via PHP Serverless Functions).
